Scanning Parameters
Configure how the extension scans for content. These are technical settings for power users who want to fine-tune performance.
Max Scan Depth: 12 ? How many levels deep the extension searches in the webpage structure. Higher values are more thorough but slower. Recommended: 8-15.

Max Text Length: 500 ? Maximum characters to analyze when detecting AI content. Lower values are faster but may miss some content. Higher values are more thorough.
Maximum text size to analyze
Scan Delay: 150ms ? Time to wait between scans when content changes. Lower values are more responsive but use more CPU. Higher values save battery.
Delay between scans
