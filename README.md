# Slop-Slurp Chrome Extension

Fine-tune search results to your liking by "slurping up" AI Overview slop, low-quality sites, and advertisements from Google Search.

## Features

- :x: Removes AI Overviews
- :broom: Hides sponsored results and advertisements
- :brain: Filters low-quality websites
- :mag_right: Settings to further tune your search results
